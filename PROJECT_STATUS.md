# HdRezka Filter: состояние и планы

## Что уже сделано

- Локальное веб-приложение запускается через `main.py` на `http://127.0.0.1:8000/`.
- Поиск по жанрам вроде `фантастика` и `детектив` идет через каталог Rezka, а не через короткие ajax-подсказки.
- По умолчанию возвращается до 100 результатов; пользователь может задать лимит от 1 до 300.
- После фильтров приложение пытается добрать результаты до заданного лимита, сканируя дополнительные страницы каталога.
- Результаты сортируются по рейтингу.
- Рейтинг для отображения и сортировки берется из IMDb.
- Есть два режима отображения:
  - карточки с постерами;
  - текстовый список без изображений.
- Есть фильтр минимального IMDb-рейтинга.
- Есть фильтр диапазона IMDb-рейтинга, например `8.0 - 5.0`.
- Есть сортировка от высокой оценки к низкой и от низкой к высокой.
- Есть включающие фильтры по жанрам и странам.
- Есть бан-листы по жанрам и странам.
- Сервер можно остановить кнопкой в интерфейсе или скриптом `stop_server.ps1`.
- Есть debug-режим: переменная окружения `HDREZKA_DEBUG=1` или чекбокс `Debug в терминал`.

## Как работают несколько тегов и банов

Поля с несколькими значениями разделяются запятыми:

```text
фантастика, детектив
США, Великобритания
аниме, ужасы
```

Включающие фильтры работают по логике `AND`: если указать `фантастика, детектив`, результат должен содержать оба жанра.

Бан-листы работают по логике `OR`: если указать `аниме, ужасы`, результат будет скрыт, если содержит хотя бы один из этих жанров.

Для стран логика такая же:

- включающие страны: должны совпасть все указанные страны;
- бан стран: скрывает результат при совпадении хотя бы одной страны.

Если поисковая строка сама является списком известных жанров, например `фантастика, детектив`, приложение собирает кандидатов из обоих каталогов и затем фильтрует результат так, чтобы остались тайтлы с указанными жанрами.

## Текущие ограничения

- IMDb-поиск может ошибиться, если у тайтла нет нормального оригинального названия или год на Rezka не совпадает с IMDb.
- Чем больше лимит результатов, тем дольше поиск: приложение открывает страницы тайтлов и добирает IMDb-рейтинг.
- Если страница тайтла на стороне Rezka временно отдает `500`, приложение сохраняет известный жанр каталога, но страна/описание могут быть пустыми.
- Сейчас каталог жанров настроен для основных жанров фильмов. Сериалы, мультфильмы и аниме можно добавить отдельными каталогами позже.

## Debug-режим

Чтобы видеть ошибки парсинга и запросов в терминале, запускай сервер так:

```powershell
$env:HDREZKA_DEBUG = "1"
& "$env:LOCALAPPDATA\Programs\Python\Python313\python.exe" main.py
```

Если сервер уже запущен из терминала, можно включить чекбокс `Debug в терминал` перед поиском.

## Как остановить сервер

Способ 1: кнопка в интерфейсе.

```text
Остановить сервер
```

Способ 2: PowerShell из папки проекта.

```powershell
.\stop_server.ps1
```

Способ 3: вручную через порт.

```powershell
netstat -ano | Select-String ':8000'
Stop-Process -Id <PID>
```
